# 健身房管理系统
**开发角色**：全栈开发

**项目周期**：2024.12.7——2025.03.28

## 项目描述
这是一套前后端分离的健身房综合管理系统，涵盖后端服务、管理员前端、用户前端三大应用，可承接健身房日常全流程运营管理工作。系统搭建了会员管理、教练管理、课程安排、商品销售、预约预订、订单处理、课程及商品评价、社区帖子发布、充值记录管理、轮播图及公告管理等完整业务模块，针对不同类型用户划分了专属操作权限与功能范围，为健身房打造了一体化的数字化管理与服务解决方案。

## 技术栈
Spring Boot、Spring Security、MyBatis Plus、PageHelper、MySQL、Druid、Vue、Vite、TypeScript、Naive UI、Pinia、UnoCSS、Element Plus、Vue Router、Vuex、Swagger、Lombok、MapStruct、FastJSON、JWT、Guava、Hutool、Maven

## 技术亮点
1. **全方位权限安全防护**：采用 MD5 盐值加密算法对用户密码等隐私数据进行加密存储，规避数据泄露后的信息窃取风险；依托 Spring Security 框架对接口请求做鉴权处理，杜绝无权限数据访问；结合 RBAC 权限模型实现菜单权限精准分配，搭配 JWT 无状态认证与验证码登录机制，构建起多层级的系统安全防护体系。
2. **标准化架构与完整数据流**：后端采用 Controller-Service-Mapper 经典分层架构，借助 MapStruct 完成 DTO、Entity、PO 各类数据对象的自动转换，配置统一的返回结果封装与全局异常处理机制；实现了从前端表单提交到后端 API 处理、数据库持久化再到前端数据展示的完整数据流闭环，提升了代码规范性与业务处理效率。
3. **全链路业务功能覆盖**：搭建了课程预约（含人数限制与余额自动扣减）、用户余额充值及支付、课程与商品评价回复、富文本内容管理、图片上传管理、充值记录追踪等实用功能，同时配置订单管理、商品 SKU 管理、社区帖子互动等模块，全方位匹配健身房的实际运营需求。
4. **高效开发与多端适配**：集成 Swagger 实现 API 文档自动生成与在线测试，通过 MyBatis Plus 简化数据库 CRUD 操作、Lombok 减少样板代码编写；借助 Druid 连接池实现数据库性能监控；管理员前端基于 UnoCSS 实现响应式布局，用户前端支持移动端适配，保障了多终端的流畅使用体验。
5. **高可维护性模块化设计**：采用模块化设计实现各业务功能解耦，配置统一的系统配置管理与完整日志记录体系；前后端通过 RESTful 风格 API 进行数据通信，既保障了系统的易维护性，也为后续功能迭代与业务拓展提供了灵活的技术支撑。
